What are the eight data types in JavaScript?
Answer: number, bigint, boolean, string, undefined, null, symbol and object.

Which data type is NOT primitive?
Answer: object.

What is the relationship between null and undefined?
Answer: Both are special data types which assign a variable as unknown or undefined.

What is the difference between single, double, and backtick quotes for strings?
Answer: Between single and double quotes there's no difference, while backtick is used for template string.

What is the term for embedding variables/expressions in a string?
Answer: { ... } (curly brackets)

Which type of quote lets you embed variables/expressions in a string?
Answer: Backtick quotes.

How do you embed variables/expressions in a string?
Answer: ` ${typeof 'text'} here and ${typeof (1)} here `

How do you escape characters in a string?
Answer: Using backslash then the char.

What is the difference between the slice/substring/substr string methods?
Answer: All these methods extract some text from the called text. slice(start, end) | substring( |start|, |end|) | substr(start, length)

What are the three logical operator and what do they stand for?
Answer: '||' stands for OR, '&&' stands for AND and '!' stands for NOT.

What are the comparison operators?
Answer: '>' stands for GRATER THAN, '<' stands for LESS THAN, '==' stands for EQUALS.

What are truthy and falsy values?
Answer: If a value become true then it's truthy, otherwise it's falsy.

What are the falsy values in JavaScript?
Answer: '', 0, null, undefined and NaN.

What are conditionals?
Answer: A piece of code which has a condition that executes some scripts whether true or not.

What is the syntax for an if/else conditional?
Answer: if ( <boolean_expression> ) {
		<something>
	   } else {
		<something_else>
	   }

What is the syntax for a switch statement?
Answer: switch( <expression> ) {
		case <choice1> :
		  <code>
		  break;
		case <choice2> :
		  <code>
		  break;
		default:
		  <code>
	   }

What is the syntax for a ternary operator?
Answer: if ( <boolean_expresion> ) ? <true> : <false>

What is nesting?
Answer: Nesting means put statement, conditional, loops, etc. within each other, respecting the execution order.
